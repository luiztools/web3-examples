# pinata-ipfs-example

A simple example written in Next.js to upload files to IPFS network using Pinata API

## How to Run
1. git clone
2. npm install
3. create .env.local file using .env.example as template
4. fill your Pinata's API key and secret in .env.local
5. npm run dev
