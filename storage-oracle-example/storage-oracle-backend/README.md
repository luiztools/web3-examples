# Storage Oracle Backend
The Node.js backend that feed the oracle.

## How to Run

1. git clone
2. deploy storage-oracle-contract at blockchain
3. cd storage-oracle-example
4. npm install
5. copy .env.example as .env
6. fill .env variables
7. npm start
