# hello-solidity
A Hello world example in Solidity using Truffle.

## Pre-requisites

1. Node.js
2. Truffle

## How to Run

1. open command line terminal
2. cd hello-solidity-truffle
3. npm install
4. sudo truffle compile
5. sudo truffle test
