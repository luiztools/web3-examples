# crud-solidity-truffle
A CRUD example in Solidity with Truffle.

## Pre-requisites

1. Node.js
2. Truffle

## How to Run

1. open command line terminal
2. cd crud-solidity-truffle
3. npm install
4. sudo truffle compile
5. sudo truffle test
